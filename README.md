# regexSearch
**本项目全部由gemini生成，本人只负责提需求和优化**

## 简要概述
chrome插件，可自定义正则表达式并且持久化存储在浏览器端，方便后续钓鱼用 

burp优化一言难尽，就没写burp插件

## 使用方法
点击start就会自动开始收集，stop就停止收集。  
规则可以自定义开启或者关闭，关闭规则就不会再匹配。  
可导出为txt。  
默认会爬取当前html中所有的js。
